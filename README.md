# N.A.V.E

Assessment executivo, guiado e gamificado baseado no NIST CSF 2.0, com foco em maturidade, priorização de lacunas e recomendações consultivas da Active Solutions.

## Estrutura

- `index.html`: entrada principal da aplicação
- `app.js`: fluxo da jornada, revisão, scoring, relatório e integrações
- `data.js`: questionário, metadados, scoring e conteúdo de UX
- `styles.css`: identidade visual e responsividade
- `assets/`: logos da Active Solutions

## Deploy

O deploy é feito via GitHub Pages pela workflow em `.github/workflows/static.yml`.
